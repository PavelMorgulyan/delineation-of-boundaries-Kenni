Задание

В среде Spyder (сборка Anaconda) на языке Python (3.х) создать проект
и подключить библиотеку scikit-image.
1. Разработать программу по загрузке цифрового изображения.
2. Подключить функцию обращения к пикселем цифрового изображения.
3. Запрограммировать формулу перевода цифрового изображения в
полутоновой формат.
4. Провести информационный поиск в Интернете по способам
применения оператора выделения границ в соответствии с вариантом.
Разобраться, включить описание в отчет (теор. часть).
5. Запрограммировать оператор выделения границ в соответствии с
вариантом. При необходимости добавить пороговое преобразование в
бинарное изображение.
6. Провести информационный поиск в Интернете и запрограммировать
работу программы распознавания фруктов в зависимости от
геометрической формы (после работы оператора выделения границ).
7. Организовать и провести эксперимент на тестовой выборке. Вычислить
ошибку первого и второго рода. Построить графики.

Пример вызова фильтра Собеля

from skimage import data, io, filters
image = data.coins()
edges = filters.sobel(image)
io.imshow(edges)
io.show()
